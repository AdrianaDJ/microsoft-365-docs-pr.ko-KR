# <a name="comparison-chart-bookings-web-app-vs-bookings-teams-app"></a>비교 차트: 예약 웹 앱 및 예약 팀 앱

팀의 예약 앱을 사용 하면 스케줄러가 주요 작업을 처리 하 고 일부 설정을 변경할 수 있습니다. 그러나 예약 웹에는 팀 앱에서 아직 사용할 수 없는 고급 기능 및 설정이 포함 되어 있습니다.

팀 앱 기능은 항상 추가 되 고 있으며이 목록을 계속 업데이트 합니다. 자세한 내용은 [FAQ](bookings-faq.md) 의 **Microsoft 팀에서 사용할 수 있는 원래 예약 웹 앱의 모든 기능** 을 참조 하세요.

| 기능 | 예약 웹 응용 프로그램 | 예약 팀 앱 |
|:---|:---|:---|
| 새 예약 일정 만들기 | 예 | 예 |
| 예약 일정에서 직원 추가 및 제거 | 예 | 예 |
| 새 약속 유형 만들기 | 예 | 예 |
| 온라인 예약 예약 | 예 | 예 |
| 비즈니스 정보 편집 | 예 | 예 |
| 조직 외부의 사용자에 대 한 게스트 사용 권한을 가진 직원 추가 | 예 | 아니요 |
| 사용자 또는 오프 라인 예약 예약 | 예 | 아니요 |
| 일정 그룹 또는 다중 고객 예약 | 예 | 아니요 |
| 비즈니스 로고 설정 | 예 | 아니요 |
| 업무 시간 설정 | 예 | 아니요 |
| 셀프 서비스 일정 관리 페이지 게시 | 예 | 아니요 |
| 고객 연락처 관리 | 예 | 아니요 |
| 직원에 대 한 휴가 설정 | 예 | 아니요 |
