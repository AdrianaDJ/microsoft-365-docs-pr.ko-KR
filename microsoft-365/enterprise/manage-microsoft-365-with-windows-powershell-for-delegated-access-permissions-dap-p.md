# <a name="how-to-manage-microsoft-365-with-windows-powershell-for-delegated-access-permissions-partners"></a>위임 된 액세스 권한 파트너로 Windows PowerShell을 사용 하 여 Microsoft 365을 관리 하는 방법

*이 문서는 Microsoft 365 Enterprise와 Office 365 Enterprise에 모두 적용됩니다.*

DAP(위임된 액세스 권한) 파트너는 Syndication 및 CSP(클라우드 솔루션 공급자) 파트너입니다. 네트워크 또는 텔레콤 공급자가 많이 있습니다. Microsoft 365 구독을 서비스 제공에 번들 합니다. Microsoft 365 구독을 판매할 때 이러한 테 넌 트을 관리 하 고 보고할 수 있도록 고객의 테 넌 트에 게 자동으로 (AOBO) 사용 권한을 부여 합니다. 이러한 작업은 Microsoft 365 관리 센터에서 수행 하기가 어렵습니다. Microsoft 365에 PowerShell을 사용 하 여 다음과 같은 관리 작업을 수행 하는 것이 훨씬 더 쉽습니다.
- 모든 고객 **TenantIds** 및 해당 도메인 나열 
- 고객의 테 넌 시 및 할당 된 라이선스의 모든 사용자 식별
> [!NOTE]
> 일부 관리 작업은 PowerShell 에서만 수행할 수 있습니다.

다음 문서에서는 신디케이션 및 CSP 파트너가 PowerShell을 사용 하 여 고객 테 넌 트를 관리 하는 방법을 보여 줍니다.
