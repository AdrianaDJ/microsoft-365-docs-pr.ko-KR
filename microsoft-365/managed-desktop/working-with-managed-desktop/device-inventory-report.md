# <a name="device-inventory-report"></a>장치 인벤토리 보고서

**장치** 보기에서 **모두 내보내기** 탭을 선택 하 여 다음 정보를 포함 하는 쉼표로 구분 된 파일을 다운로드할 수 있습니다.

- 일련 번호
- 장치 이름
- 장치 마지막 체크 인
- 관리자
- 회사
- 모델
- 르
- OS 라이선스
- OS 버전
- 소유권
- 등록 날짜
- 링 이름
- 저장소 사용 가능 GB
- 저장소-총 GB
- 사용자 UPN에 의해 등록 됨
- 할당 된 사용자
- 상태
- 보존 기간 (월)
- 프로필
- 로그인 한 사용자
