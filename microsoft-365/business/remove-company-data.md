# <a name="remove-company-data-from-devices"></a>장치에서 회사 데이터 제거

## <a name="remove-company-data"></a>회사 데이터 제거

Microsoft 365 Business을(를) 사용하여 사용자가 Microsoft 365로 보호되는 [장치](app-protection-settings-for-android-and-ios.md) 또는 [Windows PC](protection-settings-for-windows-10-devices.md)에 저장한 회사 데이터를 제거할 수 있습니다. **장치에서 회사 데이터를 제거하면 나중에 복원할 수 없습니다**. 
  
1. 전역 관리자 자격 증명을 사용하여 [Microsoft 365 Business](https://portal.office.com)에 로그인합니다. 
    
2. 관리 센터의 **장치 작업** 카드에서 **회사 데이터 제거**를 선택합니다.
    
    ![On the Devices card, choose Remove company data](media/b6fcf74b-0d7d-4e1a-894f-40f9d4a215b8.png)
  
3. **회사 데이터 제거** 창에서 데이터를 제거할 사용자를 선택하거나 검색하고 이름을 선택합니다. 
    
4. 다음 창의 **관리되는 장치** 목록에서 데이터를 제거할 장치(하나 또는 여러 개)를 선택하고 **다음**을 선택합니다. 
    
    ![On the remove comapany data pane, select the device from which you want to remove the data.](media/f3725ff9-ebdb-4c13-9523-b2df362640cf.png)
  
5. **장치에서 회사 데이터를 제거하시겠습니까?** 창에서 **제거** \> **닫기**를 선택합니다.
