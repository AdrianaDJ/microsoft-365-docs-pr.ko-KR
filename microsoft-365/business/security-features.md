# <a name="microsoft-365-business-security-features"></a>Microsoft 365 비즈니스 보안 기능

Microsoft 365 비즈니스는 Pc, 전화 및 태블릿에서 데이터 보호를 위해 단순화 된 보안 기능을 제공 합니다.
    
## <a name="microsoft-365-business-admin-center-security-features"></a>Microsoft 365 비즈니스 관리 센터 보안 기능

이러한 기능을 설정 또는 해제 하는 간단한 방법을 제공 하는 관리 센터에서 Microsoft 365 비즈니스 보안 기능을 많이 관리할 수 있습니다. 관리 센터에서 다음을 수행할 수 있습니다.
  
![Screenshot of the Devices card in the admin center](media/9982e784-dbf9-4a76-a159-bb3e2e5aa23f.png)
  
- [Android 또는 iOS 장치에 대 한 응용 프로그램 관리 설정을 지정](app-protection-settings-for-android-and-ios.md) 합니다. 
    
    이러한 설정은 설정된 된 기간 동안 후 비활성 장치에서 파일을 삭제, 작업 파일을 암호화 하, PIN, 등 사용자가 설정 되어있는지를 요구 합니다.
    
- [Windows 10 장치에 대 한 응용 프로그램 보호 설정을 지정](protection-settings-for-windows-10-devices.md) 합니다. 
    
    이러한 설정은 회사 소유의 모두에서 회사 데이터 또는 개인적으로 소유 하 고 장치에 적용할 수 있습니다.
    
- [Windows 10 장치에 대 한 장치 보호 설정을 지정](protection-settings-for-windows-10-pcs.md) 합니다. 
    
    장치를 분실 하거나 도난당, 경우에 데이터를 보호 하려면 [BitLocker](https://go.microsoft.com/fwlink/p/?linkid=871405) 암호화를 사용 하도록 설정 하 고 ransomware에 대 한 고급 보호 기능을 제공 하기 위해 [Windows 악용 Guard](https://go.microsoft.com/fwlink/p/?linkid=871404) 활성화할 수 있습니다. 
    
- [장치에서 회사 데이터 제거](remove-company-data.md)
    
    장치는 손실, 도난 또는 직원이 회사를 떠날 하는 경우 회사의 데이터를 원격으로 초기화 수 있습니다.
    
- [공장 설정으로 다시 설정 하는 Windows 10 장치](reset-devices-to-factory-settings.md) 입니다. 
    
    자신에 게 적용 되는 장치 보호 설정을 포함 하는 모든 Windows 10 장치를 다시 설정할 수 있습니다.
    
## <a name="additional-security-features"></a>추가적인 보안 기능 

Microsoft 365 비즈니스의 고급 기능을 인터넷 위협에 대 한 비즈니스를 보호 하 고 중요 한 정보를 보호 하기 위해 사용할 수 있습니다.
  
- **[Office 365 Advanced Threat Protection](https://support.office.com/article/e100fe7c-f2a1-4b7d-9e08-622330b83653)**
    
    고급 위협 보호 (ATP)를 통해 복잡 한 피싱 및 직원 또는 고객 정보를 손상 시키는 설계 ransomware 공격 으로부터 비즈니스를 보호 합니다. 기능은 다음과 같습니다.
    
  - 복잡 한 첨부 파일 검사 및 AI 기반 분석 기능을 검색 하 고 위험한 메시지를 취소 합니다.
    
  - 자동 하는 경우의 일부인 것 피싱 체계를 평가 하기 위해 전자 메일에 웹 링크의 확인 합니다. 그러면 안전 하지 않은 웹사이트에 액세스 하지 못하도록 안전 합니다.
    
- **[데이터 손실 방지 정책 개요](https://support.office.com/article/1966b2a7-d1e2-4d92-ab61-42efbb137f5e)** (DLP)입니다. 
    
    신용 카드 번호, 주민등록 번호 등의 의도 하지 않은 방지 하기 위해 다음과 같은 중요 한 정보를 자동으로 검색 하도록 DLP를 설정할 수 회사 외부의 공유 합니다.
    
- **[Exchange Online 보관](https://products.office.com/exchange/microsoft-exchange-online-archiving-email)**
    
    Exchange Online 보관 라이선스 연속 데이터 백업으로 쉽게 보관할 메시지를를 수 있습니다. 검색 또는 복원 나중에 필요한 하는 경우 삭제 된 항목을 포함 하는 사용자의 전자 메일의 모든를 저장 합니다. 또한 소송 보존, eDiscovery에 대 한 전자 메일 데이터를 유지 하기 위해 또는 규정 준수 요구 사항을 충족 하기 위해 다른 보존 정책을 사용할 수 있습니다.
    
- **[Azure Information Protection](https://go.microsoft.com/fwlink/p/?linkid=871406)**
    
    정보 보호와 같은 컨트롤이 포함 된 전자 메일 및 문서에서 중요 한 정보에 대 한 액세스를 제어 "를 수행 하지 앞 으로" 및 "복사 하지 않습니다." "Confidential"로 중요 한 정보를 분류 하 고 외부 분류 된 정보를 공유할 수 하는 방법 및 비즈니스 안에 지정할 수도 있습니다. 엔터프라이즈급 암호화 전자 메일 및 사용자의 정보를 비공개로 유지 하는 문서에 적용 하기 쉽습니다. Microsoft 365 비즈니스 [Azure 정보 보호 계획 1](https://go.microsoft.com/fwlink/p/?linkid=871407)의 모든 기능을 포함 합니다. Azure 정보 보호 클라이언트에서 추가 기능 Office 응용 프로그램에 대해 설치할 수 있습니다. 자세한 내용은 [Azure 정보 보호 클라이언트 관리자가 가이드](https://docs.microsoft.com/azure/information-protection/rms-client/client-admin-guide)를 참조 합니다.
    
- **[Azure 포털에서 Intune의 전체 기능](https://go.microsoft.com/fwlink/p/?linkid=871403)**
    
    Azure 포털의 관리 센터를 사용 하면 MacOS 장치, iPhone, 및 Windows에 대 한 고급 장치 관리와 함께 Android 장치 관리와 같은 추가적인 보안 기능을 설정 하는 Intune 액세스 (영문)를 통해 사용할 수 없는 Microsoft 365 비즈니스 관리 센터입니다.
    
다음 섹션에서는 보안에서 이러한 기능을 관리 하는 방법에 대해 설명 &amp; 준수 센터 및 Intune 관리 센터입니다. 시간에 따른 간소화 된 컨트롤은 Microsoft 365 비즈니스 관리 센터에 추가 됩니다.
  
## <a name="set-up-advanced-threat-protection-features"></a>고급 위협 보호 기능 설정

- **안전 하지 않은 첨부 파일에 대해 보호:** ATP 가상 환경에서 전자 메일 첨부 파일을 열어 악의적인 콘텐츠를 식별 하 고 결과 동작의 분석을 수행 합니다. 의도 된 (여부를 확인 하려면 일반 또는 악의적인) 계산 되는 콘텐츠, 하 고 ATP 차단 안전 하지 않은 첨부 파일의 배달 피싱 구성표 및 ransomware 감염 으로부터 시스템을 보호할 수 있습니다. 첨부 파일 보호 기능을 활성화 하려면 [Office 365 ATP 안전한 첨부 정책 설정](https://support.office.com/article/078eb946-819a-4e13-8673-fe0c0ad3a775)을 참조 하십시오.
    
- 사용자가 악의적인 링크를 클릭할 때 환경을 보호: 사용자가 해당를 클릭할 때 ATP도 검사 하 여 전자 메일에서 링크를 클릭 합니다. 링크가 안전 하지 않으면 사용자가 사이트를 방문 하 여 하지 경고가 표시 하거나 사이트 차단 된 관련 정보를 받아보십시오. 피싱을 막을 수 있습니다. [Office 365 ATP 안전한 링크 정책을 설정](https://support.office.com/article/bdd5372d-775e-4442-9c1b-609627b94b5d#reveddefaultscc) 하거나 [Office 365 ATP 안전한 링크 정책을 설정할](https://support.office.com/article/bdd5372d-775e-4442-9c1b-609627b94b5d#addemailpolscc)수 있습니다.
    
## <a name="set-up-dlp-features"></a>DLP 기능 설정

개인 식별 정보 (pii 개인) 보호 하기 위해 정책을 설정 하는 방법에 대 한 예제 [서식 파일에서 DLP 정책 만들기](https://support.office.com/article/59414438-99f5-488b-975c-5023f2254369) 를 참조 하십시오. 
  
DLP 다양 한 로캘에의 다양 한 즉시 사용 정책 템플릿을 함께 제공 됩니다. 예: 오스트레일리아 재무 데이터, 개인 정보 Act 캐나다, 미국 재무 데이터 등의 합니다. 전체 목록은 [기능는 DLP 정책 서식 파일에 포함](https://support.office.com/article/c2e588d3-8f4f-4937-a286-8c399f28953a) 을 참조 하십시오. 이러한 서식 파일의 모든 사용 가능 PII 서식 파일 예제와 비슷합니다. 
  
## <a name="set-up-email-retention-with-exchange-online-archiving"></a>Exchange Online 보관 된 전자 메일 보존 설정

 **Exchange Online 보관** 라이선스 기능 유지 하기 위해 규정 준수 및 규제 표준 전자 메일을 보존 하 여 콘텐츠 eDiscovery의 목적에 대 한 기능을 제공 합니다. 또한 소송 발생 시 위험을 줄일 수 하 고 보안 위반 사항이 나 삭제 된 항목을 복구 해야하는 경우 데이터를 복구 하는 방법을 제공 합니다. 이러한 기능을 활성화 하려면의 모든 사용자의 콘텐츠를 유지 하기 위해 소송 보존을 사용 하거나 더 많은 사용자 지정에 대 한 보존 정책을 사용 하 여 수 있습니다. 
  
**소송 보존:** 사용자의 전체 사서함을 소송 보존에 배치 하 여 삭제 된 항목을 포함 하 여 유지 하는 모든 사서함 콘텐츠를 유지할 수 있습니다. 
    
사서함에 소송 보존 관리 센터에서 시키려면:
    
1. 왼쪽된 탐색에서 **사용자** 에 게 이동 \> **활성 사용자**입니다.
    
2. 사용자 사서함을 소송 보존을 유지 하 고 선택한 사용자 창에서 **메일 설정을** 확장 하 고 **더 많은 설정** 옆에 있는 **Exchange 편집 속성**을 선택 합니다.
    
3. 사용자에 대 한 사서함 페이지에서 선택 * * 사서함 기능 * * 왼쪽된 탐색에서 하 고 **소송 보존으로 설정**아래에서 **사용 하도록 설정** 링크를 선택 합니다.
    
4. **소송 보존으로 설정** 에서 대화 상자는 소송 보존을 지정할 수 있습니다 **소송 보존 기간** 필드에 보존 기간, 비워두십시오 필드는 무한 보류를 배치 하려는 경우. 또한 메모를 추가할 수 있으며 메일 상자 소유자 보류의 소송 보존에 대 한 자세한 설명 할 수 있습니다 웹사이트를 직접 \> **저장**합니다.
    
**보존:** 특정 시간 동안 보존 또는 보존 기간의 끝에 콘텐츠를 영구적으로 삭제 하려면 사용자 지정 된 보존 정책을 설정할 수 있습니다. 자세한 내용은, [보존 정책 개요](https://support.office.com/article/5e377752-700d-4870-9b6d-12bfc12d2423)를 참조 하십시오.
## <a name="set-up-azure-information-protection-features"></a>Azure 정보 보호 기능 설정

Azure 정보 보호 (AIP)는는 조직에서 분류 하 고 필요에 따라 레이블을 적용 하 여 해당 문서 및 전자 메일을 보호 하는 데 도움이 되는 클라우드 기반 솔루션입니다. 레이블 규칙 및 사용자 또는 사용자가 권장 사항을 제공를 조합 하 여 수동으로 조건을 정의 하는 관리자가 자동으로 적용할 수 있습니다.

웹에 있는 Outlook에서 전자 메일을 보낼 때 다음과 같은 제한 사항이 적용 하는 기능이 자동으로 모든 사용자에 대해 사용 하도록 설정 됩니다.
  
- **전달 하지 않음**: 받는 사람에 게 메시지를 읽을 수는 있지만 앞으로, 인쇄, 하거나 콘텐츠를 복사할 수는 없습니다
    
- **암호화**: 전체 메시지가 암호화 됩니다. 받는 사람에 게 암호화 된 콘텐츠에 액세스 하기 전에 id를 확인 하기 위한 추가 단계를 수행 해야 하 고 암호화를 제거할 수 없습니다.
    
- **3 비밀 우편**: 전자 메일 콘텐츠 및 첨부 파일에 있지만 조직 외부의 사용자 수는 없습니다 직원 조직에 대 한 모든 권한을 제공 합니다. 데이터 소유자를 추적 하 고 언제 든 지 콘텐츠를 취소할 수 있습니다.
    
- **기밀 사항이**:이 제한은 보기, 편집 및 회신, 하지만 하지 음성 메일로 착신 전환, 인쇄 또는 데이터를 복사 하는 직원 수 있도록 하는 매우 기밀 데이터를 적용할 수 있습니다. 데이터 소유자를 추적 하 고 언제 든 지 콘텐츠를 취소할 수 있습니다.

    **관리 센터** 를 배치 하 여에 노래 수 \> **Azure Active Directory** 관리 센터의 왼쪽 탐색 창을 합니다.

2. **검색 상자**에 *Azure 정보 보호* **모든 서비스** 및 tyoe를 선택 합니다.

3. 결과가 표시 되 면 다음 즐겨찾기 하도록 **Azure 정보 보호** 하 고 나중에 찾기 쉬운 시작을 클릭 합니다.

4. **Azure 정보 보호** 선택 \> 상태 확인을 활성화 된 설정 되어 **보호 정품 인증** 하 고 확인 합니다.

### <a name="install-the-azure-information-protection-client-manually"></a>Azure 정보 보호 클라이언트를 수동으로 설치

AIP 클라이언트를 수동으로 설치 방법

 ### <a name="are-these-security-features-available-in-all-markets"></a>모든 시장에서 이러한 보안 기능을 사용할 수 있습니까?

### <a name="how-do-i-find-the-security-amp-compliance-center"></a>보안을 찾는 방법은 &amp; 준수 센터?
  
1. 관리 자격 증명을 사용 하 여 [Microsoft 365 비즈니스에 로그인](https://portal.microsoft.com/) 합니다. 
    
2. 왼쪽된 탐색, **관리 센터** 를 찾아 하 고 확장 합니다. 
    
    ![Microsoft 365 관리 센터의 왼쪽된 탐색, 관리 센터를 선택 합니다.](media/fa4484f8-c637-45fd-a7bd-bdb3abfd6c03.png)
  
3. 선택 **보안 &amp; 준수** 보안으로 이동 하려면 &amp; 준수 센터입니다. 
    
 ### <a name="how-do-i-find-the-intune-admin-center"></a>Intune 관리 센터는 어떻게 찾나요?

2. 왼쪽된 탐색, **관리 센터** 를 찾아 하 고 확장 합니다. 
    
3. Intune 관리 센터로 이동 **Intune** 선택 합니다.
