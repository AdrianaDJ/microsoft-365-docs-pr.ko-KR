# <a name="install-office-on-windows-10-during-setup"></a>설치 시 Windows 10에 Office 설치

![를 가리키는 배너 https://aka.ms/aboutM365preview 입니다.](../media/m365admincenterchanging.png)

이 설정은 기본적으로 해제되어 있습니다. 사용자가 모든 Windows 10 장치에서 최신 버전의 Office를 사용할 수 있도록 설정 하 고, 먼저 모든 필수 구성 요소가 충족 되는지 확인 합니다. [Microsoft 365 Business premium에서 Office 클라이언트 배포 준비](prepare-for-office-client-deployment.md)에 설명 된 요구 사항 외에도이 설정을 사용 하려면 Microsoft 365 business premium에 대 한 라이선스가 있어야 합니다.
  
설치 중에 기본 설정을 수락 하는 것이 좋습니다.
