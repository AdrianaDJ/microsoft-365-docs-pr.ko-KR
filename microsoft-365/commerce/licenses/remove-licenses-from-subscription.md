# <a name="remove-licenses-from-your-office-365-for-business-subscription"></a>Office 365 비즈니스 에디션 구독에서 라이선스 제거

사용자에게 라이선스가 할당된 경우 구독에서 라이선스를 제거할 수 없습니다. 현재 다른 사용자에 게 할당 된 라이선스를 제거 하려면 구독에서 라이선스를 제거 하기 전에 [사용자 로부터 라이선스를 제거](../../admin/manage/remove-licenses-from-users.md)해야 합니다.

::: moniker range="o365-worldwide"

> [!NOTE]
> 새로운 Microsoft 365 관리 센터를 사용하지 않는 경우 홈페이지 상단에 있는 **새 관리 센터 시도** 토글을 선택하여 켤 수 있습니다.

## <a name="remove-licenses"></a>Remove licenses

1. 관리 센터에서 **청구** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">제품 및 서비스</a> 페이지로 이동합니다.

2. **Products & services** 페이지에서 라이선스를 제거할 구독을 찾은 다음 **라이선스 추가/제거**를 선택 합니다.

    [라이선스 추가/제거 링크가 표시되지 않으면 어떻게 하나요?](#what-if-i-dont-see-the-addremove-licenses-link)

3. **총 라이선스** 상자에이 구독에 필요한 총 라이선스 수를 입력 한 다음 **변경 제출을**선택 합니다. 예를 들어 110개의 라이선스가 있는데 그중 5개를 제거해야 하는 경우 105를 입력합니다.

::: moniker-end

2. **구독** 페이지에서 라이선스를 제거할 구독을 선택한 다음 **라이선스 추가/제거**를 선택 합니다.

3. **총 라이선스 수** 상자에 이 구독에 필요한 총 라이선스 수를 입력한 다음 **제출**을 선택합니다. 예를 들어 110개의 라이선스가 있는데 그중 5개를 제거해야 하는 경우 105를 입력합니다.

## <a name="what-if-i-dont-see-the-addremove-licenses-link"></a>라이선스 추가/제거 링크가 표시되지 않으면 어떻게 하나요?

이 섹션에서는 **라이선스 추가/제거** 링크가 보이지 않는 이유와 이때 취할 수 있는 조치에 대해 설명합니다.
  
### <a name="a-credit-check-is-pending"></a>신용 조사가 보류 중인 경우

신용 조사가 보류 중인 경우 "신용 조사 보류 중" 메시지가 표시되고 신용 조사가 완료될 때까지 라이선스를 제거할 수 없습니다. 신용 조사가 보류 중인 경우 나중에 다시 돌아가서 신용 조사가 완료되었는지 확인합니다. 신용 조사가 완료되려면 보통 영업일 기준 최대 2일이 걸립니다.
  
신용 조사가 완료되면 **사용자** 섹션에 **라이선스 추가/제거** 링크가 표시됩니다. 해당 하는 경우 [Office 365 for business 구독에서 라이선스 제거](#remove-licenses-from-your-office-365-for-business-subscription)로 이동 합니다.
  
### <a name="you-activated-the-subscription-using-a-product-key"></a>제품 키를 사용하여 구독을 활성화한 경우

25자의 제품 키를 사용하여 구독을 구입하고 정품 인증한 경우 "선불"이라는 텍스트가 표시됩니다. 제품 키를 사용하여 구독을 선불 구입한 경우 요금이 이미 지불되었으므로 라이선스를 제거할 수 없습니다. 그러나 구독을 갱신할 때 추가 라이선스를 제거할 수 있습니다.
  
### <a name="you-bought-your-subscription-through-a-partner"></a>파트너를 통해 구독을 구입한 경우

CSP 또는 파트너를 통해 구독을 구매한 경우에는 라이선스를 제거할 수 없습니다. 도움을 요청 하려면 해당 CSP에 문의 하거나 VLSC (볼륨 라이선스 서비스 센터) 링크를 사용 하 여 파트너에 게 문의 하세요.
  
## <a name="what-you-need-to-know-about-removing-licenses-from-users-in-office-365-for-business"></a>비즈니스용 Office 365에서 사용자 라이선스를 제거하는 데 대해 알아야 할 사항

- 전역 관리자 또는 사용자 관리 관리자 여야 합니다. 자세한 내용은 [관리 역할 정보](../../admin/add-users/about-admin-roles.md)를 참조 하세요.

- 기존 사용자 계정에 라이선스를 추가하려면 이 단계를 수행합니다. [사용자 계정을 추가하고 동시에 라이선스를 할당하는 방법을 알아보세요](../../admin/add-users/add-users.md).
