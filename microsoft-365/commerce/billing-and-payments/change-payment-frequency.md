# <a name="change-your-payment-frequency"></a>결제 빈도 변경

구독을 구입할 때 월별 또는 연간 대금 청구를 선택 합니다. 구독에 대 한 요금 청구 빈도를 변경 하려면 다음 단계를 사용 합니다.

1. 관리 센터에서 <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">제품</a> **청구** \> 페이지로 이동 합니다.

2. 변경 하려는 구독을 찾습니다. **청구** 섹션에서 **매월 청구** 되거나 **매년 청구**옆에 있는 **편집**을 선택 합니다.

3. 대금을 청구할 빈도에 대 한 옵션을 선택한 다음 **변경을**선택 합니다.
