# <a name="microsoft-365-security-preview-features"></a>Microsoft 365 보안 미리 보기 기능

**적용 대상:**
- Microsoft 위협 방지



Microsoft 365 보안은 새로운 기능 개선 사항을 포함하기 위해 지속적으로 업데이트됩니다.

Microsoft 365 보안 미리 보기 릴리스의 새 기능에 대해 알아보고 미리 보기 환경을 설정하여 출시 예정 기능을 먼저 사용해보세요.

일반 공급되는 새 기능에 대한 자세한 내용은 [Microsoft 위협 방지의 새 기능](mtp-whats-new.md)을 참조하세요.
