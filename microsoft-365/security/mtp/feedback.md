# <a name="provide-feedback-on-microsoft-365-defender"></a>Microsoft 365 Defender에 의견 제공

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**적용 대상:**
- Microsoft 365 Defender

사용자 의견을 활용 하면 고급 공격 으로부터 환경을 보호 하는 데 도움이 됩니다. 사용자 의견을 제공 하 여 경험, 상대 및 요청을 공유 합니다.

1. 포털의 어느 부분에서 나 **의견 제공** 을 선택 합니다. 

    ![사용자 의견 단추 이미지](../../media/feedback.png)

2. 환경을 평가 하 고 좋아요를 설정 하거나 개선할 수 있는 위치에 대 한 세부 정보를 제공 합니다. 사용자 의견에 대해 연락 하도록 선택할 수도 있습니다. 

3. **전송** 을 선택합니다.
