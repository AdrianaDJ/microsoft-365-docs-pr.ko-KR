# <a name="generate-a-test-alert-in-your-microsoft-365-defender-evaluation-lab"></a>Microsoft 365 Defender 평가 랩에서 테스트 경고 생성  

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**적용 대상:**
- Microsoft 365 Defender

Microsoft 365 Defender evaluation lab 설치 및 구성을 완료 했으므로 위협 공격 시뮬레이션을 실행 하 여 검색, 경고 상관 관계 및 통합 된 보안 솔루션의 자가 치유 된 치료 기능을 경험할 수 있습니다.  

1. 로그온 대상 https://security.microsoft.com
2. 홈 페이지에서 **시뮬레이션** 이라는 타일을 찾습니다.  Microsoft 365 Defender에서 테스트 인시던트를 만드는 단계는 연습 가이드를 참조 하십시오.
