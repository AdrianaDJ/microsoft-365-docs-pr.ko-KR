# <a name="microsoft-threat-protection-prerequisites"></a>Microsoft 위협 방지 필수 구성 요소

**적용 대상:**
- Microsoft 위협 방지

[!include[Prerelease information](prerelease.md)]

라이선스, 하드웨어 및 소프트웨어 요구 사항 및 기타 구성 설정을 사용하여 Microsoft 365 보안을 실행하고 사용하는 방법에 대해 알아봅니다.

## <a name="licensing-requirements"></a>라이선스 요구 사항
Microsoft 365 보안을 사용하려면 다음 라이선스 중 하나가 필요합니다.

- Microsoft 365 E5 
- Office 365 E5, Enterprise Mobility + Security E5 및 Windows E5

[Microsoft 365 Enterprise 페이지](https://www.microsoft.com/en-us/microsoft-365/enterprise)에서 이 라이선스를 받을 수 있습니다.

### <a name="check-your-existing--licenses"></a>기존 라이선스 확인
기존 라이선스를 보려면 [admin.microsoft.com](https://admin.microsoft.com/)에서 Microsoft 365 관리 센터로 이동합니다. 관리 센터에서 **청구** > **라이선스**로 이동합니다.

라이선스 정보를 보려면 [Azure AD에서](https://docs.microsoft.com/azure/active-directory/users-groups-roles/directory-assign-admin-roles#available-roles) **청구 관리자** 또는 **전역 독자** [역할이 있어야 합니다. 액세스 문제가 발생하는 경우 전역 관리자에게 문의하세요.  

## <a name="browser-requirements"></a>브라우저 요구 사항
Microsoft 365 보안 센터에 대한 액세스는 브라우저를 통해 수행됩니다. Internet Explorer 및 Microsoft Edge가 지원됩니다. 모든 HTML5 호환 브라우저도 지원됩니다.
