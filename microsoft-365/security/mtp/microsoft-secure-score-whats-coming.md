# <a name="whats-coming-to-microsoft-secure-score"></a>Microsoft Secure Score의 새로운 기능

Microsoft [보안 점수를 더](microsoft-secure-score.md) 정확하게 대표하고 사용 편의성을 높이기 위해 당사는 거의 필요에 따라 점점을 변경합니다. 점수와 가능한 최대 점수가 변경될 수 있습니다.

## <a name="proposed-changes"></a>제안된 변경 내용

이러한 변경 사항은 라이선스에 따라 곧 Microsoft 보안 점수에 적용될 수 있습니다. 최근 변경 내용에 대한 자세한 내용은 [Microsoft Secure Score의 새로운 기능을 확인하나요?](microsoft-secure-score-whats-new.md)

- "레거시 인증 차단을 위한 정책 사용"이라는 Azure Active Directory 개선 작업에 대해 최대 점수를 7개에서 8로 변경합니다.
