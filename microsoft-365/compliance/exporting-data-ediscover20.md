# <a name="export-case-data-in-advanced-ediscovery"></a>Advanced eDiscovery에서 사례 데이터 내보내기

다음과 같은 세 가지 방법으로 검토 집합에서 데이터를 내보낼 수 있습니다.

**다운로드:** 브라우저를 사용 하 여 소수의 기본 파일 집합을 다운로드 합니다. 이 방법은 작은 데이터 집합을 내보내는 가장 빠른 방법입니다. 이 메서드는 네이티브 파일 이름을 유지 합니다. 자세한 내용은 [검토 집합에서 문서 다운로드](download-documents-from-review-set.md)를 참조 하세요.

**내보내기:** 내보낼 데이터를 사용자 지정 합니다. 여기에는 PDF 파일에 저장 된 파일 메타 데이터, 네이티브 파일, 텍스트 파일 및 redacted 문서를 내보내는 작업이 포함 됩니다. 내보낸 데이터를 준비한 후에는 **내보내기** 탭에서 로컬 컴퓨터로 데이터를 다운로드할 수 있습니다. 자세한 내용은 [문서를 검토 집합에서 내보내기](export-documents-from-review-set.md) 를 참조 하세요.

**다른 검토 집합에 추가:** 한 검토 집합에서 다른 검토 집합으로 데이터를 복사 합니다. 자세한 내용은 [한 검토 집합의 데이터를 다른 검토 집합에 추가](add-data-to-review-set-from-another-review-set.md)를 참조 하세요.

> [!NOTE]
> Microsoft 365에서는 데이터가 해시 되며 확인을 위해 이러한 해시가 출력 파일에 제공 됩니다. 이 기능은 수집 통계, 부하 집합 보고서 및 내보내기 보고서 (내보내기 로드 파일 포함)와 같은 감사 로그 및 보고 기능을 통해 보완 됩니다.
