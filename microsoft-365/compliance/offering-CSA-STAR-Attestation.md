## <a name="csa-star-attestation-overview"></a><span data-ttu-id="c44fe-105">CSA STAR 증명 개요</span><span class="sxs-lookup"><span data-stu-id="c44fe-105">CSA STAR self-assessment overview</span></span>

<span data-ttu-id="c44fe-106">CSA(Cloud Security Alliance)는 STAR(Security, Trust & Assurance Registry: 보안, 트러스트 및 보증 등록)라고 하는 클라우드 서비스 공급자(CSP)가 무료 및 공개적으로 액세스할 수 있는 등록 프로그램을 유지 관리합니다.</span><span class="sxs-lookup"><span data-stu-id="c44fe-106">The Cloud Security Alliance (CSA) maintains the Security, Trust & Assurance Registry (STAR), a free, publicly accessible registry where cloud service providers (CSPs) can publish their CSA-related assessments.</span></span> <span data-ttu-id="c44fe-107">STAR는 CSA CCM(Cloud Controls Matrix)의 제어 목표에 맞춰진 세 가지 보장 수준으로 구성됩니다.</span><span class="sxs-lookup"><span data-stu-id="c44fe-107">STAR consists of three levels of assurance aligned with control objectives in the CSA Cloud Controls Matrix (CCM).</span></span> <span data-ttu-id="c44fe-108">(CCM은 클라우드 고객이 클라우드 서비스의 전반적인 보안 위험을 평가하는 데 도움이 되도록 16개 도메인의 기본 보안 원칙을 다룸):</span><span class="sxs-lookup"><span data-stu-id="c44fe-108">(The CCM covers fundamental security principles across 16 domains to help cloud customers assess the overall security risk of a cloud service.):</span></span>

- <span data-ttu-id="c44fe-109">수준 1: STAR 자체 평가</span><span class="sxs-lookup"><span data-stu-id="c44fe-109">Level 1: STAR Self-Assessment</span></span>
- <span data-ttu-id="c44fe-110">수준 2: STAR 증명, STAR 인증, C-STAR 평가(제3자의 감사를 기준으로 함)</span><span class="sxs-lookup"><span data-stu-id="c44fe-110">Level 2: STAR Attestation, STAR Certification, and C-STAR Assessment (which are based on audits by third parties)</span></span>
- <span data-ttu-id="c44fe-111">수준 3: STAR 연속 모니터링(프로그램 요구 사항은 계속 CSA에서 개발 중에 있음)</span><span class="sxs-lookup"><span data-stu-id="c44fe-111">Level 3: STAR Continuous Monitoring (program requirements are still under development by CSA)</span></span>

<span data-ttu-id="c44fe-112">STAR 증명에는 CCM 기준에 따라 SOC 2 유형 2 감사를 기준으로 클라우드 공급자의 보안 상태를 엄격하게 독립적으로 감사하는 작업이 포함됩니다.</span><span class="sxs-lookup"><span data-stu-id="c44fe-112">STAR Attestation involves a rigorous independent audit of a cloud provider’s security posture based on a SOC 2 Type 2 audit with CCM criteria.</span></span> <span data-ttu-id="c44fe-113">STAR 증명에 대한 클라우드 공급자의 서비스를 평가하는 독립 감사자는 CPA(Certified Public Accountant)여야 하며, 이는 CCSK(Cloud Security Knowledge)에 CSA 인증서를 보유하는 데 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="c44fe-113">The independent auditor that evaluates a cloud provider’s offerings for STAR Attestation must be a certified public accountant (CPA) and is required to have the CSA Certificate in Cloud Security Knowledge (CCSK).</span></span>  
  
<span data-ttu-id="c44fe-114">SOC 2 유형 2 감사는 AICPA(American Institute of Certified Public Accountants) 트러스트 서비스 원칙과 조건을 기반으로 합니다. 예를 들어 보안, 가용성, 기밀성, 프로세싱 무결성, CCM의 조건이 포함됩니다.</span><span class="sxs-lookup"><span data-stu-id="c44fe-114">A SOC 2 Type 2 audit is based on American Institute of Certified Public Accountants (AICPA) Trust Services Principles and Criteria, including security, availability, confidentiality, and processing integrity, and the criteria in the CCM.</span></span> <span data-ttu-id="c44fe-115">STAR 증명은 Microsoft 클라우드 서비스에서 SOC 2 컨트롤의 디자인 적합성과 운영 효율성에 대한 감사자가 알아낸 내용을 제공합니다.</span><span class="sxs-lookup"><span data-stu-id="c44fe-115">STAR Attestation provides an auditor’s findings on the design suitability and operating effectiveness of SOC 2 controls in Microsoft cloud services.</span></span> <span data-ttu-id="c44fe-116">그 목표는 앞서 설명한 AICPA 조건과 CCM에 설정된 요구 사항을 모두 충족하는 것입니다.</span><span class="sxs-lookup"><span data-stu-id="c44fe-116">The objective is to meet both the AICPA criteria mentioned above and requirements set forth in the CCM.</span></span>
