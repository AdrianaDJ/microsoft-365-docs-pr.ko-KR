# <a name="additional-steps-to-export-system-generated-log-data"></a>시스템 생성 로그 데이터를 내보낼 수 있는 추가 단계

**Kaizala**&nbsp;&nbsp;&nbsp;Kaizala 관리 포털을 사용하면 조직의 제품 및 서비스 사용 현황 데이터를 내보내고 Excel 기능을 사용하여 특정 사용자의 데이터를 필터링할 수 있습니다. 자세한 지침은 [Kaizala에서 조직 데이터 내보내기](https://support.office.com/article/export-user-data-in-kaizala-150ec214-a070-4e8a-8509-82f46d84bbb6)를 참조하세요.

**Office 로밍 서비스**&nbsp;&nbsp;&nbsp;Office 로밍은 Office 테마, 사용자 지정 사전, 언어 설정, 개발자 모드, 자동 고침과 같은 Office 관련 설정을 저장하는 서비스입니다. 이 데이터를 내보내는 방법에 대한 지침은 [Office 로밍 서비스에서 데이터 내보내기](https://support.office.com/article/manage-gdpr-data-subject-requests-with-the-dsr-case-tool-in-the-office-365-security-compliance-center-preview-ce9eb942-3589-42cb-88fd-1576ecb09c5c?storagetype=stage#o365datamoreinfo)를 참조하세요. 
 
**작업 공간 분석**&nbsp;&nbsp;&nbsp;데이터 로그 내보내기 도구는 작업 공간 분석 보고서 실행 권한이 있는 조직의 사용자에 대한 사용 현황 데이터를 제공합니다. 작업 공간 분석은 Office 365 데이터에서 파생된 필명화된 데이터를 계산하고 저장하여 성능을 향상합니다. 이 필명화된 데이터를 사용자가 사용할 수 있도록 만들고 도움이 필요한 [Microsoft 지원](https://support.microsoft.com/contactus/)에 문의하세요.

**Yammer**&nbsp;&nbsp;&nbsp;Yammer 관리 센터에서 사용자의 계정 활동 데이터를 내보낼 수 있습니다. 사용자 데이터를 내보내면 사용자의 계정 활동 데이터가 포함된 전자 메일 메시지를 받게 됩니다. 선택하면 이 정보를 사용자에게 제공할 수 있습니다. 자세한 지침은 [Yammer Enterprise: 개인 정보](https://support.office.com/article/eae49f12-4661-4ba5-aa72-01248f0709bf)를 참조하세요.
