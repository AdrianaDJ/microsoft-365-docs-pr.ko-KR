# <a name="manage-messages-in-message-center"></a>메시지 센터에서 메시지 관리

메시지 센터 게시물은 새로운 기능과 변경 된 기능에 대 한 중요 한 알림입니다. 이 문서에서는 조직의 변경 관리에 도움을 주는 메시지에서 수행할 수 있는 작업에 대 한 자세한 정보를 제공 합니다.
  
## <a name="read-a-message"></a>메시지 읽기

메시지 센터로 이동 하려면 [관리 센터로](https://go.microsoft.com/fwlink/p/?linkid=2024339)이동한 후 **상태**  >  **메시지 센터**를 선택 합니다. 

메시지 제목을 선택 하 여 읽기 창에서 엽니다. 모든 메시지에는 변경 내용의 요약, 변경에 영향을 주는 방법에 대 한 설명, 변경에 대 한 조직을 준비 하는 방법에 대 한 정보, 그리고 대부분의 경우 더 자세히 알아볼 수 있는 추가 정보 링크가 포함 되어 있습니다. 메시지가 만료 된 후에도 추가 정보를 계속 사용할 수 있습니다.

## <a name="give-feedback-on-a-post"></a>게시물에 대 한 의견을 제공 합니다.

메시지 센터에서 메시지를 선택 하 여 세부 정보를 확인할 수 있습니다.

메시지에 대 한 피드백을 제공 하려면 **Like** 또는 **Dislike** 아이콘 중 하나를 선택 하 고 표시 되는 텍스트 상자에 의견을 제공 합니다. 개인 정보를 제공 하지 않습니다. 원하는 경우 **확인을 선택 하 여이 사용자 의견에 대해 문의** 한 후 **보내기를**선택 합니다.

## <a name="share-a-message"></a>메시지 공유

다른 사용자가 작업 해야 하는 메시지 표시 전자 메일을 통해 메시지 내용을 모든 사용자와 공유할 수 있습니다.
  
1. 메시지를 선택 하 여 열고 **공유**를 선택 합니다.
  
2. 메시지를 공유 하려면 전자 메일 주소를 각각 콜론으로 구분 하 여 입력 합니다. 전자 메일 주소를 개인 및 그룹으로 보낼 수 있습니다. 필요한 경우 메시지 복사본을 전자 메일 (기본 전자 메일 주소로 이동)으로 받거나 개인 메시지를 추가 하 여 받는 사람에 게 더 많은 컨텍스트를 제공 하도록 선택할 수 있습니다.
  
3. **공유** 를 선택 하 여 전자 메일을 보냅니다.
    
## <a name="get-a-link"></a>링크 가져오기

다른 관리자와 함께 추가 작업을 수행 해야 변경 내용을 확인 하 고 조치를 취할 수 있습니다. 사용자를 해당 메시지에 직접 연결 하는 것과 같이 전자 메일 또는 인스턴트 메시징에 공유 하기 위한 링크를 생성할 수 있습니다. 링크를 공유한 사람은 조직에 대해 Microsoft 365의 관리자 이거나 연결 된 메시지에 액세스할 수 없습니다.

::: moniker range="o365-worldwide"

1. 메시지를 선택 하 여 엽니다.

2. **클립보드에 링크 복사** (링크) 아이콘을 선택 합니다.

3. 웹 페이지에서 클립보드에 액세스할 수 있습니다. 직접 링크가 클립보드에 복사 됩니다.

4. Ctrl + V를 사용 하거나, 마우스 오른쪽 단추를 클릭 하 고 **붙여넣기** 를 선택 하 여 링크를 삽입 합니다.

::: moniker-end

::: moniker range="o365-germany"

1. 메시지를 선택 하 여 엽니다.

2. **More** \> 작업 모음에서 추가 **링크 가져오기를** 선택 합니다.

3. 웹 페이지에서 클립보드에 액세스할 수 있습니다. 직접 링크가 클립보드에 복사 됩니다.

4. Ctrl + V를 사용 하거나, 마우스 오른쪽 단추를 클릭 하 고 **붙여넣기** 를 선택 하 여 링크를 삽입 합니다.

::: moniker-end

::: moniker range="o365-21vianet"

1. 메시지를 선택 하 여 엽니다.

## <a name="read-and-unread-states"></a>읽음 및 읽지 않음 상태

메시지 센터에서 읽지 않은 메시지는 굵게 표시 됩니다. 메시지를 열면 해당 메시지가 읽은 것으로 표시 됩니다. 메시지를 읽지 않은 상태로 표시할 수 있습니다.

::: moniker range="o365-worldwide"

1. 메시지 센터의 기본 페이지에서 메시지를 선택 합니다.

2. 메시지 맨 위에 **읽지 않은 상태로 표시** (봉투) 아이콘을 선택 합니다.
  
::: moniker-end

::: moniker range="o365-germany"

1. 메시지를 선택 하 여 엽니다.

2. **More** \> **읽지 않은 것으로 표시를**선택 합니다.

3. 메시지를 닫습니다.

::: moniker-end

::: moniker range="o365-21vianet"

1. 메시지를 선택 하 여 엽니다.

2. **More** \> **읽지 않은 것으로 표시를**선택 합니다.

3. 메시지를 닫습니다.

::: moniker-end

## <a name="archive-and-restore"></a>보관 및 복원

사용자에 게 해당 하는 메시지가 표시 되지 않거나 이미 처리 된 경우 메시지를 보관 하 여 메시지 센터 보기에서 해당 메시지를 제거할 수 있습니다. 메시지 센터에 표시 되는 보기는 사용자 계정에 따라 달라 지 며 보기에서이를 보관 하면 다른 관리자에 게 영향을 주지 않습니다.

::: moniker range="o365-worldwide"

- 메시지 센터의 기본 페이지에서 메시지를 선택 하 고 메시지 목록 위에 **보관** 을 선택 합니다.

- 메시지를 열고 메시지 아래쪽에서 **보관** 을 선택 합니다.

보관 된 메시지를 다시 받아야 하나요? 문제 없어요.
  
1. 메시지 센터 위쪽에서 **보관 된 메시지** 탭을 선택 합니다. 보관 된 메시지 목록이 표시 됩니다. 

2. 메시지를 선택 하 고 **복원을**선택한 다음 메시지가 활성 메시지로 복원 됩니다.

::: moniker-end

::: moniker range="o365-germany"

메시지를 보관 하는 방법에는 두 가지가 있습니다.
  
- 메시지 ID 오른쪽에 표시 되는 X를 선택 합니다. 메시지가 목록에서 제거 됩니다. Or,

- 메시지를 열고 메시지 위쪽에서 **보관** 을 선택 합니다.

해제 된 메시지를 다시 받아야 하나요? 문제 없어요.
  
1. **Views (보기** ) 드롭다운 메뉴에서 **보관 된 메시지** 필터를 선택 합니다. 보관 된 메시지 목록이 표시 됩니다.

2. 메시지 ID 오른쪽에 있는 **복원을** 선택 하면 메시지가 활성 메시지로 복원 됩니다.

::: moniker-end

::: moniker range="o365-21vianet"

메시지를 보관 하는 방법에는 두 가지가 있습니다.
  
- 메시지 ID 오른쪽에 표시 되는 X를 선택 합니다. 메시지가 목록에서 제거 됩니다. Or,

- 메시지를 열고 메시지 위쪽에서 **보관** 을 선택 합니다.

보관 된 메시지를 다시 받아야 하나요? 문제 없어요.
  
1. **Views (보기** ) 드롭다운 메뉴에서 **보관 된 메시지** 필터를 선택 합니다. 보관 된 메시지 목록이 표시 됩니다.

2. 메시지 ID 오른쪽에 있는 **복원을** 선택 하면 메시지가 활성 메시지로 복원 됩니다.

::: moniker-end

메시지 센터에 대 한 개요는 [Microsoft 365의 메시지 센터](message-center.md)를 참조 하세요. 또는 메시지 센터 게시물에 대 한 기계 번역을 사용 하도록 언어 기본 설정을 지정 하는 방법에 [대 한 자세한 내용은 message center 게시물에 대 한 언어 번역](language-translation-for-message-center-posts.md)를 참조 하세요. 실시간 서비스 상태 정보 및 메시지 센터 통신을 가져오는 다른 방법을 프로그래밍 하려는 경우에는 [Microsoft 365 서비스 통신 API 개요](https://go.microsoft.com/fwlink/p/?linkid=848507)를 참조 하세요.
