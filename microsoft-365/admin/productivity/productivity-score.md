# <a name="microsoft-productivity-score-preview"></a>Microsoft 생산성 점수 (미리 보기)

Microsoft 365에서는 조직이 비즈니스 목표를 충족할 수 있도록 하는 생산성 솔루션을 제공 합니다. 생산성 점수는 작업 수행 방식으로 변환할 수 있는 이러한 솔루션을 사용 하기 위한 정보를 제공 합니다.다음과 같은 내용이 포함 되어 있습니다. 

- **가시성** 에 따라 직원 환경 및 기술 경험 (곧 출시 예정) 범주에 걸쳐 현재 점수 형식으로 작동 하는 방식을 이해 하는 데 도움이 됩니다. 
- 향상 된 환경을 사용할 수 있는 기회를 식별 하기 위한 **정보** 
- 모든 사용자가 최상의 작업을 수행할 수 있도록 기술 및 시스템을 업데이트 하기 위한 **작업** 입니다. 

점수 및 정보 제공은 두 가지 범주로 나뉩니다.  

- **직원 환경:** Microsoft 365에서 사용자가 콘텐츠를 공동으로 작업 하 고, 어디서 든 업무를 수행 하 고, 통신 스타일 이해 (곧 방문) 및 모임 문화권 개발 (출시 예정)을 통해 생산적이 고 효율적인 인력을 만드는 방법을 보여줍니다. 

- **기술 환경** (출시 예정): 일반적인 지원 센터 문제를 사전에 수정 하 고 PC 시작 시간 및 네트워크를 개선 하 여 앱이 제대로 작동 하는지 확인 하는 등의 장치 환경을 최적화할 수 있도록 지원 합니다.  

## <a name="how-it-works"></a>작업 방법

### <a name="areas-of-scoring"></a>점수 매기기 영역 

각 점수 범주 내에서 생산성 점수는 여러 지역에서 작업 변환에 대 한 정보를 제공 합니다. 직원 환경에 포함 되는 영역은 다음과 같습니다.

- 콘텐츠 공동 작업  
- 외부에서 작업 
- 통신 스타일 이해 (출시 예정) 
- 모임 문화권 개발 (출시 예정) 

### <a name="scoring-user-actions"></a>사용자 작업 점수 매기기 

각 영역 내에서 조직이 생산성이 높은 조직으로 변환 될 수 있도록 하는 방법을 나타내는 연구 기반 주요 작업을 측정 합니다. 각 영역에 대해 지난 28 일 이내에 이러한 작업을 수행 하는 사용자 의%를 계산 합니다. 

직원 환경 점수는 모든 지역의 평균 점수로 맞게 조정 된 됩니다. Microsoft는 시간에 따른 직원 환경 및 기술 경험에 더 많은 영역을 추가 합니다. 

### <a name="products-included-in-the-productivity-score"></a>생산성 점수에 포함 된 제품 

생산성 점수에는 현재 주요 Microsoft 365 워크 로드 OneDrive, SharePoint, Word, Excel, PowerPoint, OneNote, Outlook, Yammer, 팀, Skype의 신호가 포함 되어 있습니다. 

점수가 매일 업데이트 되며 지난 28 일 이내에 완료 된 사용자 작업 (현재 날짜 포함)을 반영 합니다.

## <a name="access-and-required-permissions"></a>액세스 및 필수 권한 

직원 환경에서는 여러 사용자가 포함 된 비즈니스 계획을 위한 Microsoft 365를 구독 해야 합니다. 

Microsoft 생산성 점수에 액세스할 수 있는 권한을 부여 하려면 다음 역할이 있어야 합니다. 

- 전역 관리자 
- Exchange 관리자 
- SharePoint 관리자 
- 비즈니스용 Skype 관리자 
- Teams 관리자 
- 전역 읽기 권한자 
- 보고서 구독자 

왼쪽 탐색 창에서**생산성 점수** **보고서** > 를 선택 하 여 Microsoft 365 관리자 홈의 환경에 액세스할 수 있습니다.

## <a name="interpreting-productivity-score"></a>생산성 점수 해석 

### <a name="learn-how-your-organization-works"></a>조직의 작동 방식 알아보기 

생산성 점수 홈 페이지에는 벤치 마크로 보완 된 각 점수 범주 내의 영역에 대 한 기본 정보를 제공 하 여 현재 점수 및 역사를 백분율로 표시 합니다. 

![생산성 점수 홈 페이지](../../media/pslanding.png)

1.**생산성 점수가** 백분율 단위로 <numerator> / <denominator> 나열 되며 절대 점수 (분자) 및 최대 가능한 점수를 볼 수도 있습니다.  
1. 이 피벗을 사용 하면 포커스를 구하려는 점수 범주를 선택할 수 있습니다. 미리 보기에서 **직원 환경을**볼 수만 있고, 선택 영역은 **기술 환경을**포함 하도록 곧 확장 됩니다. 
1. **피어 벤치 마크** 를 사용 하면 현재와 같은 성과를 조직과 비교할 수 있습니다. 벤치 마크 측정값은 비슷한 조직의 집합 내에서 발생 하는 평균 측정값으로 계산 됩니다. 이 집합은 사용 중인 사용자, 지역, 소유 하 고 있는 라이선스 유형, 산업, Microsoft 365 또는 Office 365 내의 tenure가 비슷한 조직으로 구성 됩니다. 
1. **점수 범주** 섹션에서는 항목별 벤치 마크에 대 한 생산성 성과를 분석 하 여 보여 줍니다.
1. **점수 기록** 에서는 지난 6 개월 동안 각 범주의 점수가 이동 된 방식을 표시 합니다.
1. **콘텐츠의 공동** 작업에 대 한 주요 파악 사항에는 점수에 기여 하는이 영역의 벤치 마크가 포함 된 관련 측정값이 표시 됩니다. **세부 정보 보기** 를 선택 하 여 영역 세부 정보 페이지를 표시 합니다.
1. **사용** 에 대 한 기본적인 이해에는이 영역의 점수에 기여 하는 벤치 마크가 포함 된 관련 측정값이 나와 있습니다. **세부 정보 보기** 를 선택 하 여 영역 세부 정보 페이지를 표시 합니다.  

### <a name="insights-to-identify-opportunities-to-improve-experiences"></a>환경을 개선 하기 위한 기회 식별을 위한 정보 

각 영역에 대해 기본 통찰력의 **세부 정보 보기** 를 선택 하 여 추가 insights, 관련 연구 및 작업을 표시 하는 영역 정보 페이지를 보고 점수를 높입니다. .

모든 영역 세부 정보 페이지는 다음 구조를 따릅니다. 

- 왼쪽 위: 홈 페이지에서 기본 통찰력이 보존 됩니다.
- 왼쪽 아래: 상황별 작업을 사용한 insights를 지원 합니다.  
- 권한: 영역 및 생산성 간의 관계를 보여 주는 교육 기관 연구

또한 모든 정보에는 다음 요소가 포함 됩니다. 

- 헤더: 주요 요지 또는 각 통찰력에 대해 추적 되는 결과입니다.
- 요약: 조직의 생산성에 대 한 통찰력의 측정값이 중요 한 이유에 대 한 자세한 설명입니다. 
- 데이터 시각화: 시각적으로 벤치 마크를 사용 하는 경우의 quantifies에 대 한 측정값을 고려 하 여 조직의 직위를 파악 합니다.
- 동작: 이해를 돕기 위해 원하는 결과를 보다 효율적으로 구동 하 여 점수를 높일 수 있는 상황별 작업입니다.

### <a name="area-detail-page--collaborating-on-content"></a>영역 세부 정보 페이지-콘텐츠 공동 작업 

![콘텐츠 페이지의 Collaborationg](../../media/pscollab.png)

콘텐츠 페이지 공동 작업 생산성 점수 홈 페이지의 주요 통찰력을 포함 합니다. 여기에는 다음과 같은 구성 요소가 있습니다.

1. 헤더: Quantifies 콘텐츠를 공동으로 작업 하는 사용자의 비율 (주요 결과)입니다.
1. Body: 공동 작업의 효율성이 향상 되는 방식에 대 한 자세한 정보입니다.
1. 시각화:이 영역에 대 한 점수를 제공 하는 각 요소가 벤치 마크를 통해 시각적으로 quantifies.

    - **독자**: Quantifies 사용자가 Onedrive 및 sharepoint 사용 가능 사용자의 기본 내에서 클라우드에서 파일에 액세스 하거나 다운로드 합니다.
    - **작성자**: Quantifies 사용자가 Onedrive 및 sharepoint 사용 가능 사용자의 기본 내에서 클라우드 (Onedrive 및 sharepoint)의 파일을 수정, 업로드, 동기화, 체크 인, 복사 또는 이동 하는 작업을 수행 합니다.
    - **공동**작업: Quantifies 사용자는 onedrive 및 sharepoint 사용 가능 사용자의 기본 내에서 클라우드 (Onedrive 및 sharepoint)의 파일에 대해 공동 작업을 수행 합니다. 두 사용자가 한 명의 공동 작업자 중 하나에 게는 한 명 이상의 작업 창 내에서 Word, Excel, PowerPoint, OneNote 또는 PDF 클라우드 문서를 읽거나 수정한 후 해당 문서가 작성 되거나 편집 된 것입니다.

1. **It가 중요 한 이유** 는 클라우드 파일에서 생산성에 대 한 리서치 연결 공동 작업을 요약 한 것입니다. 자세한 내용은 공동 작업을 conducive 하는 방식에 대 한 추가 컨텍스트를 제공 하는 연구 문서를 **Forrester에서 연구를 참조** 하세요.
1. **여러 파일에 대 한 공동** 작업을 지원 하 여 조직의 사용자가 서로 다른 파일에서 공동으로 작업 하는 방법을 quantifies. 
1. 사용자가 내부 및 외부 공유를 구분 하는 동시에 클라우드에서 콘텐츠를 공유 하는 경우 quantifies **클라우드 파일 공유** 에 대 한 통찰력을 지원 합니다.
1. 전자 메일을 통한 파일 공유에 대 한 자세한 정보 **를 quantifies.**

공동 작업용으로 고려 되는 파일 형식은 Word/Excel/PowerPoint/OneNote/PDF입니다.

### <a name="area-detail-page--working-anywhere"></a>영역 세부 정보 페이지-외부에서 작업 

![모든 위치에서 사용 페이지](../../media/psflexibility.png)

외부에서 작업 중 페이지는 데스크톱, 모바일 및 웹 인 두 개 이상의 플랫폼에서 적어도 한 명의 생산성 앱을 사용 하는 사용자 (백분율 및 절대값)를 quantifies 합니다. 사용자는 더 많은 앱과 플랫폼을 사용 하기 위해 점수가 높다는 것을 볼 수 있습니다. 고려 되는 앱은 Outlook, 팀, Word, Excel, PowerPoint, OneNote, Yammer, Skype입니다. 의견 기반은 사용자가 enterprise, Exchange, Yammer, Skype 또는 팀에 대해 Microsoft 365 앱을 사용 하도록 설정 하는 것입니다.

1. **It가 중요 한 이유** 는 여러 플랫폼의 사용 현황을 조사 하 여 생산성을 높일 수 있는 방법을 제공 하는 것입니다. 자세한 **내용은 Forrester from** conducive에서 생산성 향상에 대 한 자세한 컨텍스트를 제공 하는 연구 문서로 이동 합니다.
1. **외부에서 전자 메일** 에 대 한 자세한 정보 quantifies outlook에서 활성 상태인 사용자 내에서 플랫폼을 통해 outlook을 사용 하 여 조직의 사용자 수를 파악 합니다.
1. **외부에서 채팅** 에 대 한 자세한 정보 quantifies 조직의 사용자가 팀에서 활성 상태인 기본 사용자 내에서 플랫폼을 통해 팀을 사용 하는 방법을 설명 합니다.
1. Office 문서에 액세스 하는 방법에 대 한 자세한 내용은 outlook, Excel, PowerPoint 또는 OneNote에서 활성 상태인 사용자의 quantifies에서 **언제 어디서 나** Word, Excel, Powerpoint 및 onenote를 사용 하는 방법을 조직의 사용자에 게 활용 하는 방법을 지원 합니다.

### <a name="actions-for-update-skills-and-systems"></a>업데이트 기술 및 시스템에 대 한 작업 

Actionability을 승격 하기 위해 지원 되는 각 정보에는 직원의 업무를 변환 하는 데 도움이 되는 인라인 작업과 구성 변경 또는 홍보 캠페인이 포함 될 수 있습니다. 이러한 통화를 클릭 하면 권장 되는 작업 패널 옵션에서 고려해 야 할 수 있습니다. 현재 **콘텐츠에 대 한 공동** 작업의 작업이 사용 되도록 설정 됩니다.

![Office 문서 공동 작업 페이지입니다.](../../media/psactions1.png)

1. **보기 작업** 은 콘텐츠 공동 작업과 관련 된 모든 작업을 나열 하는 창으로 이동 합니다.
1. 공동 **작업을 수행 하는 사용자에** 게 공동 작성 및 @mentions 같은 항목과 공동 작업을 수행 하는 방법을 안내 하는 비디오 목록을 설명 합니다. 
1. **외부 공유 설정 사용자 지정** 은 외부 공유에 대 한 개요를 포함 하는 페이지 이며, 공유 대상 및 사용자가 공유 하는 사람에 따라 공유할 때 수행 되는 작업을 설명 합니다.  
1. **첨부 파일을 공유 링크로 바꾸기** 더 나은 공동 작업을 위해 전자 메일을 통해 첨부 파일 대신 OneDrive 링크를 공유 하는 방법을 설명 하는 페이지로 이동 합니다.

실행 단추 중 하나를 선택 하면 오른쪽에 각 작업에 대 한 요약과 문서 페이지에 대 한 링크가 있는 **권장 작업** 창이 열립니다.

![권장 작업 창입니다.](../../media/psacttions2.png)
