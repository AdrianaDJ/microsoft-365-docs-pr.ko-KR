# <a name="microsoft-productivity-score"></a>Microsoft 생산성 점수 

생산성 점수는 조직에서 Microsoft 365와 이를 지원하는 기술 경험을 활용하는 방법에 대한 인사이트를 활용하여 작업 수행 방법을 변화시키는 데 도움이 됩니다. 점수는 직원 및 기술 경험 측정치에 대한 조직의 성과를 반영하고 귀사와 비슷한 조직과 점수를 비교합니다.

점수에는 다음이 포함됩니다.

- **메트릭** 은 귀하가 Microsoft 365 제품을 사용하여 직원이 여러 플랫폼에서 공동 작업하고 의사소통하고 작업하는 방식을 확인하는 데 도움이 됩니다.
- 데이터 **인사이트** 는 직원의 생산성과 만족도를 개선할 수 있는 기회를 식별하는 데 도움이 됩니다.
- 취할 수 있는 **권장 조치** 는 조직의 모든 사용자가 최상의 작업을 할 수 있도록 Microsoft 365 제품을 효과적으로 사용하는 데 도움을 줄 수 있습니다.

다음과 같은 두 가지 영역에서 데이터, 인사이트 및 추천 사항을 제공합니다. 

- **사용자 경험:** 사용자가 콘텐츠에 대해 공동 작업하는 방식, Microsoft 365 제품을 사용하여 의사소통하는 방식, 여러 플랫폼에서 Microsoft 365를 사용하는지 여부를 측정합니다. 

    사용자가 온라인으로 공동 작업할 때 시간을 절약하고 모든 장치에서 작업할 수 있는 자유로 생산성과 만족도가 높아지기 때문에 이러한 인사이트를 제공합니다. 사용자가 유연하게 의사소통할 수 있으면 사용자를 더 효율적으로 만들고 더 나은 관계를 형성할 수 있게 하여, 조직이 더 통합되게 할 수 있습니다. 증거는 [Forrester 보고서](https://vc2prod.blob.core.windows.net/vc-resources/TEIStudies/TEI%20of%20Microsoft%20365%20E5%20-%20Oct%202018.pdf)를 참조하세요.

- **기술 경험:** 팀의 생산성은 Microsoft 365를 효율적으로 사용하는 것뿐만 아니라 안정적이고 잘 수행되는 기술에 달렸습니다. [엔드포인트 분석](https://aka.ms/endpointanalytics)을 사용하면 하드웨어 및 소프트웨어의 성능 및 상태 문제로 인해 사용자의 생산성이 어떻게 영향을 받을 수 있는지를 이해할 수 있습니다.권장 조치는 이러한 문제를 해결하는 데 도움이 됩니다. Microsoft 365 네트워크 연결 인사이트는 조직의 연결 문제를 해결하는 데 도움이 됩니다.

개요와 필수 구성 요소 세부 사항에 대한 자세한 내용은 [엔드포인트 분석이란](https://docs.microsoft.com/mem/analytics/overview)을 참조하세요. Microsoft 365 네트워크 연결 인사이트에 대한 자세한 내용은 [네트워크 연결 개요](https://docs.microsoft.com/microsoft-365/enterprise/microsoft-365-networking-overview)를 참조하세요.
  

## <a name="how-the-score-is-calculated"></a>점수 계산 방법

생산성 점수는 사용자 및 기술 경험 범주에 대한 총 점수를 기준으로 합니다. 각 범주는 총 100포인트의 동등한 가중치를 가집니다. 가능한 최대 생산성 점수는 800점입니다.

### <a name="score-categories"></a>점수 범주 

- 커뮤니케이션(100포인트)
- 모임(100포인트)
- 콘텐츠 공동 작업(100포인트)
- 팀워크(100포인트)
- 이동성(100포인트)
- 엔드포인트 분석(100포인트)
- 네트워크 연결(100포인트)
- Microsoft 365 앱 상태(100포인트)
- **가능한 총 합계 = 800포인트**
 
 각 점수 범주에서 사용자가 Microsoft 365 제품을 사용하여 여러 플랫폼에서 공동 작업하고, 의사소통하고, 작업하는 방식에 대한 지표인 주요 활동의 패턴을 식별합니다. Microsoft는 주요 활동에 대한 28일 및 180일 보기를 제공합니다. 또한 점수 계산에 포함되지는 않지만 해결할 수 있는 기본 동작과 설정을 식별하는 데 도움을 주는 중요한 지원 메트릭도 제공합니다.

### <a name="products-included-in-productivity-score"></a>생산성 점수에 포함된 제품 

생산성 점수는 Exchange, SharePoint, OneDrive, Teams, Word, Excel, PowerPoint, OneNote, Outlook, Yammer 및 Skype의 데이터를 포함합니다.

점수는 매일 업데이트되며 지난 28일(현재 날짜 포함)에 완료된 사용자 작업을 반영합니다.


## <a name="pre-requisites"></a>필수 구성 요소 

사용자 경험 데이터의 경우 비즈니스용 Microsoft 365 또는 Office 365 enterprise 구독이 필요합니다. 테넌트에 대한 엔드포인트 분석 데이터의 경우 구독에 Microsoft Intune을 추가해야 합니다. Intune은 장치와 앱을 관리하여 조직의 데이터를 보호하는 데 도움이 됩니다. Intune을 설치하면 Intune 환경에서 엔드포인트 분석을 켤 수 있습니다. Microsoft Intune에 대해 자세히 알아보세요. 
> [!NOTE]
> 생산성 점수 기능을 사용하기 위해 Workplace Analytics에 대한 라이선스는 필요하지 않습니다.

조직의 생산성 점수를 보려면 다음 역할 중 하나가 있어야 합니다. 

- 전역 관리자 
- Exchange 관리자
- SharePoint 관리자 
- 비즈니스용 Skype 관리자 
- Teams 관리자 
- 전역 읽기 권한자 
- 보고서 읽기 권한자 

**보고서** > **생산성 점수** 의 Microsoft 365 관리자 홈에서 경험에 액세스할 수 있습니다.

## <a name="interpreting-productivity-score"></a>생산성 점수 해석 

생산성 점수 홈페이지에는 총 점수, 점수 기록, 각 범주에 대한 주요 인사이트가 표시됩니다.

:::image type="content" source="../../media/prodscore-landing.png" alt-text="보고서의 생산성 점수 페이지.":::

**점수** 는 퍼센트 값과 포인트로 표시됩니다. 분자의 포인트와 분모의 가능한 최대 포인트를 볼 수 있습니다.

**피어 벤치마크** 를 사용하여 귀하와 비슷한 조직과 점수를 비교할 수 있습니다. 사용자 경험 범주의 피어 벤치마크는 유사한 조직 집합 내의 측정값 평균으로 계산됩니다. 조직 집합은 라이선스가 있는 사용자, 라이선스 유형, 산업, Microsoft 365 사용 기간 수가 유사한 귀하 지역의 조직으로 구성됩니다. 

엔드포인트 분석 피어 벤치마크에는 장치 시작 성능 및 모든 테넌트의 집계된 중간값 값을 기반으로 권장되는 소프트웨어 구성에 대한 대상이 포함됩니다.

네트워크 연결의 경우 권장되는 벤치마크는 80포인트입니다.

**점수 분석** 섹션에서는 사용자 및 기술 경험 영역별로 벤치마크와 함께 생산성 점수 분석을 제공합니다.

점수 기록에서는 지난 6개월 동안 각 범주의 점수가 어떻게 변했는지를 표시합니다.

**사용자 경험** 과 **기술 경험** 영역에는 해당 영역의 범주에 대한 주요 인사이트가 포함되어 있습니다. 각 범주를 클릭하여 세부 인사이트를 확인할 수 있습니다.
