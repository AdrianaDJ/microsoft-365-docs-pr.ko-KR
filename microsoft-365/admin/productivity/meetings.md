# <a name="meetings--people-experiences"></a>모임 – 사용자 환경

생산성 점수는 Microsoft 365 및이를 지 원하는 기술 환경을 통해 조직의 디지털 변환에 대 한 정보를 제공 합니다. 조직의 점수가 사용자 및 기술 경험 측정법을 반영 하며,이에 해당 하는 조직에 대 한 벤치 마크와 비교할 수 있습니다. 모임 범주는 사용자가 경험 하는 측정값의 일부입니다. 자세한 내용은 [생산성 점수 개요](productivity-score.md) 를 확인 하 고 [Microsoft의 개인 정보 취급 방침](https://privacy.microsoft.com/privacystatement)을 읽어 보세요.

## <a name="why-your-organizations-meetings-score-matters"></a>조직의 모임 점수가 중요 한 이유

사용자가 아이디어를 탐색 하 고, 문제를 해결 하 고, 의사 결정을 내리는 데 도움이 되는 모임은 조직의 생산성을 위한 기본 기둥입니다. 조사에서는 사용자가 온라인 모임 도구를 효과적으로 사용할 때 매주 최대 104 분까지 저장 됨을 의미 합니다. [증거](https://vc2prod.blob.core.windows.net/vc-resources/TEIStudies/TEI%20of%20Microsoft%20365%20E5%20-%20Oct%202018.pdf)를 참조 하세요.

## <a name="how-we-calculate-the-meetings-score"></a>모임 점수를 계산 하는 방법

이 범주에 대 한 주요 메트릭이 포함 된 환경에서 주요 통찰력을 제공 합니다. 그런 다음 아래에서 설명 하는 점수 매기기 프레임 워크를 사용 하 여 조직의 점수를 계산 합니다.

### <a name="primary-insight"></a>기본 통찰력

Microsoft 팀은 Outlook 일정과 통합 되며, 회의를 보다 효율적이 고 효과적으로 수행할 수 있는 기능을 제공 합니다. 기본 통찰력은 조직에 있는 모든 Microsoft 팀 온라인 모임에 대해 조사 합니다. 기본 규정은 팀 회의에 대 한 모범 사례 중 하나 이상을 따르는 모임 수를 추적 하기도 합니다.

:::image type="content" source="../../media/primaryinsights-meetings.png" alt-text="모범 사례를 포함 하는 모임에 대 한 기본 정보":::

1. **머리글:** 회의 중에 비디오 또는 화면을 공유 하는 이전 28 일 이내에 보관 되는 Microsoft 팀의 온라인 모임 비율을 보여 줍니다.
2. **본문:** 모임 중에 비디오 또는 화면 공유를 사용 하는 등의 모범 사례를 진행 하는 방법에 대 한 자세한 정보를 제공 하 여 모임을 보다 효율적으로 만들 수 있습니다.
3. **시각화 (현재 상태):**

      - 이 가로 막대형 차트에서 파란색 (색) 부분은 머리글에 표시 되는 백분율을 나타냅니다.
      - 분수 (분자/분모)는 헤더에 표시 되는 백분율을 계산 하는 데 사용 됩니다.
         - 분자: 조직의 사용자를 포함 하 여 비디오 또는 화면 공유를 사용 하는 온라인 Microsoft 팀 회의 수입니다.
         - 분모: 조직의 사용자를 포함 하 여 지난 28 일간 개최 된 온라인 Microsoft 팀원의 수입니다.
      - 키 메트릭의 피어 벤치 마크 값도 백분율로 표시 됩니다.
1. **리소스에 연결:** 도움말 콘텐츠를 보려면이 링크를 선택 합니다.

#### <a name="trend-visualization-of-the-primary-insight"></a>주요 통찰력 추세 시각화

다음 차트에서는 기본 통찰력의 키 메트릭 분자와 분모의 추세를 보여 줍니다. 즉, 비디오 또는 화면 공유와 같은 모범 사례와 최근 180 일 동안 개최 되는 총 온라인 Microsoft 팀원 수를 보여 주는 온라인 Microsoft 팀원의 수를 표시 합니다. 꺽은선형 차트의 각 데이터 요소는 지난 28 일 동안의 작업을 집계 한 것입니다.

:::image type="content" source="../../media/number-meeting-over-time.png" alt-text="지난 28 일 동안의 온라인 팀 회의 수를 보여 주는 차트입니다.":::

### <a name="scoring-framework"></a>성과 구조 평가

조직의 모임 점수는 조직의 온라인 Microsoft 팀 모임이 지난 28 일 동안의 모범 사례를 따르는 정도를 측정 합니다. 조직에서 모임에 참석 하는 사용자 수와 모임 시간을 기준으로 가중치가 부여 됩니다.

## <a name="explore-more-about-meetings-in-your-organization"></a>조직의 모임에 대해 자세히 알아보기

또한에 지에 대 한 지원 정보를 제공 하 여 조직의 사용자가 집계를 수행 하는 방식을 이해할 수 있습니다. 이러한 추가 메트릭은 생산성 점수에 직접적으로 영향을 주지 않지만 디지털 변환의 일부로 작업 계획을 만드는 데 도움이 될 수 있습니다.

### <a name="breakdown-of-how-many-meetings-follow-best-practices"></a>모범 사례를 따르는 모임 수 분석

:::image type="content" source="../../media/videouse-meetings.png" alt-text="모임 사용자 환경에 대 한 기본 정보를 보여 주는 차트":::

1. **머리글:**  주요 통찰력과 점수를 고려 하 여 비디오 모범 사례를 사용 하는 온라인 Microsoft 팀 회의의 비율을 강조 합니다.
2. **본문:** Hrovides 정보를 활용 하 여 모임 중에 이러한 방법을 사용 하는 경우의 가치에 대해 설명 합니다. 
3. **시각화:** 시각화의 분석 결과는 다음과 같은 모범 사례를 각각 따라 온라인 Microsoft 팀 모임이 진행 되는 범위를 나타내기 위한 것입니다. 
    - **비디오**: 색이 지정 된 부분과 분수는 비디오가 켜져 있는 온라인 Microsoft 팀 모임의 비율을 나타냅니다. 분수는 다음을 통해 생성 됩니다. 
        - 분자: 한 명 이상의 참가자가 설정 했던 지난 28 일 이내에 개최 되는 온라인 Microsoft 팀 모임입니다. 
        - 분모: 지난 28 일 이내에 조직에 보관 된 총 온라인 Microsoft 팀 모임 수입니다. 
    - **화면 공유**: 색이 지정 된 부분 및 분수는 사용자가 화면 공유 기능을 사용 하는 온라인 Microsoft 팀원의 백분율을 나타냅니다. 분수에는 다음이 포함 됩니다. 
        - 분자: 지난 28 일 동안 개최 되는 온라인 Microsoft 팀원에 게 해당 화면을 공유 하는 참가자가 한 명 이상 있습니다.
        - 분모: 지난 28 일 동안 조직에서 수행 된 총 온라인 Microsoft 팀 모임 수입니다.


### <a name="distribution-of-time-spent-by-people-in-meetings"></a>모임에서 사용자가 사용한 시간 분포

:::image type="content" source="../../media/percentageofpeopleusingteams.png" alt-text="최대 20 시간 동안 팀 회의에 참가 하는 사용자의 비율을 보여 주는 차트":::

1. **머리글:** 지난 28 일의 활동을 기준으로 평균적으로 주당 20 시간 이상 온라인 Microsoft 팀 회의에 참가 하는 조직 내 사용자의 백분율을 표시 합니다.
2. **본문:** 모임 모범 사례를 사용 하 여 모임에 참여 하 고 생산성을 향상 시킬 수 있는 가치에 대해 자세히 설명 합니다.
3. **시각화:** 지난 28 일 동안의 모임에 소요 된 평균 시간을 기준으로 조직의 사용자에 대 한 보기를 제공 합니다. 각 범주에 대해 제공 되는 정보는 다음과 같습니다.
      - **총 모임 참석자:** 지난 28 일 동안의 평균 모임 시간 범위를 기반으로 회의에 참가 한 조직의 사용자 수를 표시 합니다. 예를 들어, 6-10 hours 범주는 지난 28 일 이내에 일주일 동안의 평균으로 회의에 참가 한 사용자 수를 나타냅니다.
      - **비디오가 있는 모임의 참석자:** 각 범주에 대해 지난 28 일 동안 비디오를 사용 하 여 조직에 있는 사용자의 수를 보여 줍니다.
      - **화면 공유가 있는 모임의 참석자:** 각 범주에 대해 지난 28 일 동안 화면 공유가 포함 된 모임에 참가 한 사용자의 수를 보여 줍니다.

### <a name="distribution-of-meeting-length-by-type"></a>유형별 모임 길이 분포

:::image type="content" source="../../media/distribution-meetinglength.png" alt-text="모임 길이의 분포를 보여 주는 차트입니다.":::

1. **머리글:** 30 분 이내에 지난 28 일 동안 최근 예약 되지 않은 인스턴트 온라인 Microsoft 팀 모임이 표시 되는 백분율을 보여 줍니다.
2. **본문:** 문제를 신속 하 게 해결 하기 위해 즉석 모임을 사용 하는 값에 대 한 정보를 제공 합니다.
3. **시각화:** 지난 28 일 이내에 조직에서 발생 한 즉석 및 예약 모임의 길이 분포를 분 단위로 제공 합니다. 배포 시 각 모임은 1-15 분, 16-30 분, 31-60 분, 60 분 이상으로 분류 됩니다.

    > [!NOTE]
    > 예약 된 모임에는 사용자의 일정에 표시 된 모든 모임이 포함 됩니다. 이러한 통화에는 &quot; &quot; Microsoft 팀 채널에서 모임 시작 기능을 사용 하기 시작한 모임 뿐 아니라 1:1 및 그룹 통화를 비롯 한 통화가 포함 됩니다.


### <a name="use-of-different-meeting-types"></a>다양 한 모임 유형 사용

:::image type="content" source="../../media/percentparticipation-meetingtypes.jpg" alt-text="서로 다른 모임 유형에 참여 하는 비율을 보여 주는 차트":::

1. **머리글:** 이전 28 일 동안 시간이 30 분 미만인 인스턴트 온라인 Microsoft 팀원의 백분율을 강조 합니다.
2. **본문:** &quot; &quot; Microsoft 팀 채널 기능에서 모임 시작 사용의 값에 대 한 정보를 제공 합니다.
3. **시각화:** 온라인 Microsoft 팀 회의에 참가 하 고 있는 사용자가 참가 하 고 있는 모임 유형을 표시 합니다. 각 모임 유형은 가로 막대로 표시 되며, 색이 지정 된 부분 및 분수는 다음을 나타냅니다.
    - **즉각적인 1:1 통화**:
        - 분자: 지난 28 일 이내에 1:1 통화에 참가 한 사용자 수입니다.
        - 분모: 지난 28 일 동안 온라인 Microsoft 팀 회의에 참가 한 사용자의 수입니다.
   - **인스턴트 그룹 통화**:
        - 분자: 지난 28 일 동안의 그룹 통화에 참가 한 사용자 수입니다.
        - 분모: 지난 28 일 동안 온라인 Microsoft 팀 회의에 참가 한 사용자의 수입니다.
   - **채널의 빠른 모임 시작**:
        - 분자: &quot; &quot; 지난 28 일 동안의 Microsoft 팀 채널 (즉석 모임)에서 모임 시작 기능을 사용 하는 사용자 수입니다.
        - 분모: 지난 28 일 동안 온라인 Microsoft 팀 회의에 참가 한 사용자의 수입니다.
    - **예약 된 일회용 모임:**
        - 분자: 지난 28 일 내에 일정에 따라 일회용 온라인 Microsoft 팀 모임이 참석 하는 사용자 수입니다.
        - 분모: 지난 28 일 동안 온라인 Microsoft 팀 회의에 참가 한 사용자의 수입니다.
    - **예약 된 되풀이 모임:**
        - 분자: 지난 28 일 동안 일정에서 되풀이 모임에 참가 하는 사용자의 수 (예약 됨)입니다.
        - 분모: 지난 28 일 동안 온라인 Microsoft 팀 회의에 참가 한 사용자의 수입니다.
