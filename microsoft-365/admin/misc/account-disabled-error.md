# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>웹용 Outlook에서 계정을 사용할 수 없도록 설정 하는 중 오류 발생

웹에서 Outlook (이전의 Outlook Web App)을 열려고 할 때 **계정이 사용 하지 않도록 설정** 된 경우 `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` 관리자가 웹용 outlook에 대 한 액세스를 사용 하지 않도록 설정 했을 수 있습니다.

관리자는 [사서함에 대해 사용 또는 사용 안 함 Outlook Web App](https://docs.microsoft.com/exchange/recipients-in-exchange-online/manage-user-mailboxes/enable-or-disable-outlook-web-app)항목의 단계를 수행 하 여이 오류를 해결할 수 있습니다.
