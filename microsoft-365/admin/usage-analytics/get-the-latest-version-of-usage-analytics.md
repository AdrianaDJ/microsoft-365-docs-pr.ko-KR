# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Microsoft 365 사용 현황 분석의 최신 버전 가져오기

서식 파일 앱은 1 년에 여러 번 새 데이터 또는 새로운 시각화로 새로 고칠 수 있습니다. 기존 인스턴스는 계속 작동 하지만 최신 버전을 가져오려면 새 인스턴스를 만들고 모든 사용자 지정 내용을 새 인스턴스에 적용 해야 합니다. [Enable Microsoft 365 usage analytics](enable-usage-analytics.md)를 참조 하세요.
